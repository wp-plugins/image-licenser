Easy tagging of images with a CreativeCommons-license by the new shortcode cc. By using RDFa search engines can detect the license for the images.

== Description ==

This plugin provides an easy way to tag embedded images with a CreativCommons license. This uses RDFa, to enable search engines finding your images even if the user filters for CC licenses.

Please leave your feature requests and bug notifications by commenting on the [plugin page](http://www.raphael-mack.de/wp-image-licenser/) as I don't read the comments here very often...

This plugin reuses code and ideas from the embedded-video-with-link plugin by Stefan He&szlig; (jovelstefan).

== Installation ==

Copy the subfolder "image-licenser" with all included files into the "wp-content/plugins" folder of WordPress. Then go to the menu "Administration" > "Plugins" and activate it. 

== Configuration ==

The configuration options are simple and do not need explanation. See the administration panel of the plugin in your WP admin frontend.
